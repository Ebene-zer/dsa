# Data Structures and Algorithms


### Structures, ADTs, and Applications
---
1. **Stack**: A stack is a linear data structure that follows the Last In First Out (LIFO) principle.
- **ADT**: Last-in, First-out
- **Core operations**:
  - **Push**: Adds an element to the top of the stack.
  - **Pop**: Removes and returns the top element of the stack.
  - **Peek**: Returns the top element without removing it.
  - **IsEmpty**: Checks if the stack is empty.
  - **Size**: Returns the number of elements in the stack.
- **Application**: Stacks are used in various applications like expression evaluation, browser history, undo & redo operations, and backtracking algorithms, and function call management in programming languages.

**Edge cases to consider**:
- Underflow - Popping from an empty stack.
- Underflow - Peeking at an empty stack.
- Overflow - Pushing to a full stack (in case of a fixed-size implementation).
